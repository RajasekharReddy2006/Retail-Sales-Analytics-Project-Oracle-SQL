# 🛒 Retail Sales Analytics Database (Oracle SQL)

## 📌 Project Overview

The **Retail Sales Analytics Database** is an Oracle SQL project designed to demonstrate database design, SQL programming, and business data analysis. The project simulates a retail store management system by maintaining customer, product, employee, order, payment, and return information.

This project is suitable for SQL beginners, college students, and anyone preparing for Oracle SQL interviews.

---

## 🎯 Project Objectives

- Design a relational database using Oracle SQL.
- Implement tables with Primary Keys and Foreign Keys.
- Insert sample data into all tables.
- Practice SQL queries from basic to advanced.
- Analyze retail sales data using SQL.
- Build a strong SQL portfolio project for GitHub.

---

## 🗂 Database Tables

| Table Name | Description |
|------------|-------------|
| Category | Stores product category details |
| Customer | Stores customer information |
| Product | Stores product details |
| Employee | Stores employee information |
| Orders | Stores customer order details |
| Order_Details | Stores products ordered in each order |
| Payment | Stores payment information |
| Return_Order | Stores returned products and refund details |

3. Run `01_Create_Tables.sql`.
4. Run `02_Insert_Data.sql`.
5. Run `03_SQL_Queries.sql` to practice and test SQL queries.
